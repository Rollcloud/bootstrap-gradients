# Bootstrap Gradients

## Setup

1.  Install and activate latest LTS version of Node
    ```
    nvm install lts
    nvm use 22.19.0
    ```
2.  Install packages
    ```
    npm install
    ```
3.  Run website
    1. Run website in development mode
       ```
       npm run dev
       ```
    2. Compile website for production environment
       ```
       npm run build
       ```

## Lint and format

To format source files, run

```
npx prettier --write .
```
